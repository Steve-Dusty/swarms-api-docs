**Swarm Type**: `AutoSwarmBuilder`

## Overview

The AutoSwarmBuilder swarm type automatically analyzes your task and generates an optimal swarm configuration. Instead of manually designing agent teams and selecting swarm architectures, AutoSwarmBuilder intelligently determines the best structure, creates appropriate agents, and orchestrates their collaboration. This is ideal for users who want powerful multi-agent capabilities without deep expertise in swarm design.

Key features:
- **Automatic Agent Generation**: Creates specialized agents based on task analysis
- **Intelligent Architecture Selection**: Chooses the optimal swarm type for your task
- **Adaptive Configuration**: Adjusts parameters based on task complexity
- **Zero Configuration**: Get started with just a task description

## Use Cases

- Rapid prototyping of multi-agent solutions
- Tasks where optimal swarm architecture is unclear
- Users new to multi-agent systems
- Complex tasks requiring automatic decomposition
- Dynamic workloads with varying requirements

```go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io/ioutil"
    "net/http"
)

type SwarmConfig struct {
    Name        string `json:"name"`
    Description string `json:"description"`
    SwarmType   string `json:"swarm_type"`
    Task        string `json:"task"`
    MaxLoops    int    `json:"max_loops"`
}

func main() {
    API_BASE_URL := "https://api.swarms.world"
    API_KEY := "your_api_key_here"

    swarmConfig := SwarmConfig{
        Name:        "Auto-Generated Research Team",
        Description: "Automatically build optimal swarm for research task",
        SwarmType:   "AutoSwarmBuilder",
        Task:        "Conduct a comprehensive analysis of the current state of quantum computing, including technical challenges, major players, recent breakthroughs, and potential applications in the next 5 years",
        MaxLoops:    1,
    }

    jsonData, _ := json.Marshal(swarmConfig)

    req, _ := http.NewRequest("POST", API_BASE_URL+"/v1/swarm/completions", bytes.NewBuffer(jsonData))
    req.Header.Set("x-api-key", API_KEY)
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        fmt.Printf("Error: %v\n", err)
        return
    }
    defer resp.Body.Close()

    body, _ := ioutil.ReadAll(resp.Body)
    fmt.Printf("Response: %s\n", string(body))
}
```

## Best Practices

- Provide clear, detailed task descriptions for better auto-generation
- Use the `rules` parameter to guide agent creation with specific requirements
- Review generated configurations for production use cases
- Start with AutoSwarmBuilder for prototyping, then refine manually if needed
- For repeated tasks, consider saving the generated configuration for direct reuse
