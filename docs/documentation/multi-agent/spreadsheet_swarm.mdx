**Swarm Type**: `SpreadSheetSwarm`

## Overview

The SpreadSheetSwarm is designed for processing tabular data, performing financial calculations, and handling batch data operations. Multiple agents work together to analyze rows, columns, or cells of data, making it ideal for financial analysis, data validation, report generation, and complex spreadsheet-like operations at scale.

Key features:
- **Tabular Data Processing**: Handle data in rows, columns, and cells
- **Parallel Calculations**: Process multiple data points simultaneously
- **Financial Analysis**: Built for financial modeling and calculations
- **Batch Operations**: Efficient processing of large datasets
- **Data Validation**: Multiple agents can validate and cross-check data

## Use Cases

- Financial statement analysis and reporting
- Portfolio risk assessment and calculations
- Data transformation and ETL pipelines
- Batch invoice or transaction processing
- Comparative analysis across multiple metrics
- Budget planning and forecasting

## Best Practices

- Use low temperature (0.1-0.3) for accurate numerical calculations
- Design agents with specific analytical focuses for comprehensive coverage
- Include a synthesizer agent to combine individual analyses
- Structure data clearly in the task description
- For large datasets, consider breaking into smaller batches
- Use consistent data formats to enable cross-agent analysis
