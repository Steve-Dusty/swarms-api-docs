**Swarm Type**: `GroupChat`

## Overview

The GroupChat swarm type enables multiple agents to engage in collaborative, conversational problem-solving. Agents interact with each other in a dynamic discussion format, building on each other's ideas, challenging assumptions, and working together to arrive at comprehensive solutions. This architecture mimics human team discussions and is ideal for tasks requiring brainstorming, collaborative analysis, or consensus-building.

Key features:
- **Dynamic Conversation**: Agents engage in back-and-forth dialogue
- **Collaborative Problem-Solving**: Multiple perspectives contribute to solutions
- **Idea Building**: Agents can expand on or refine each other's contributions
- **Natural Workflow**: Mimics human team discussions and meetings

## Use Cases

- Brainstorming sessions for creative solutions
- Collaborative code review and debugging
- Strategic planning and decision-making
- Complex problem analysis requiring multiple expert perspectives
- Team-based research and analysis

## Best Practices

- Design agents with complementary expertise for richer discussions
- Use higher temperature values (0.6-0.8) to encourage creative dialogue
- Set appropriate `max_loops` to allow sufficient rounds of discussion
- Include agents that can challenge assumptions for more robust outcomes
- Use system prompts that encourage collaborative behavior and building on others' ideas
